First launch

Opening an unsigned alpha build.

The macOS build opens normally. The Windows build is not yet code-signed, so SmartScreen will warn you the first time — the steps below are only needed once.

macOS

  1. Open the DMG and drag PartSpan to Applications.
  2. Double-click to launch. The app is signed and notarized by Apple — no security bypass needed.

中文：打開 DMG，把 PartSpan 拖進「應用程式」，直接雙擊開啟即可。App 已完成 Apple 簽章與公證。

Windows

  1. Run the installer EXE.
  2. When SmartScreen shows "Windows protected your PC", click More info, then Run anyway.

中文：SmartScreen 出現「Windows 已保護您的電腦」時，點「其他資訊」再點「仍要執行」。
