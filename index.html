First launch

Opening an unsigned alpha build.

The alpha is not yet code-signed. Your OS will warn you the first time. This is expected — the steps below are only needed once.

macOS

  1. Open the DMG and drag PartSpan to Applications.
  2. Right-click (or Control-click) PartSpan.app and choose Open.
  3. If macOS still blocks it: open System Settings → Privacy & Security, scroll down, and click Open Anyway next to the PartSpan message.
  4. If you see "PartSpan is damaged", run this once in Terminal: xattr -cr /Applications/PartSpan.app

中文：安裝後對 PartSpan.app 按右鍵選「打開」。若仍被擋，到「系統設定 → 隱私權與安全性」點「強制打開」。 出現「已損毀」訊息時，在終端機執行一次 xattr -cr /Applications/PartSpan.app。

Windows

  1. Run the installer EXE.
  2. When SmartScreen shows "Windows protected your PC", click More info, then Run anyway.

中文：SmartScreen 出現「Windows 已保護您的電腦」時，點「其他資訊」再點「仍要執行」。
