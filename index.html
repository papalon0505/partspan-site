Simple operating flow

Three steps to organize the first product BOM.

Create local workspace

PartSpan stores product, part, BOM, AVL, inventory, and audit data in a local SQLite database.

Free Edition runs locally.

The first release is distributed as a local Node.js app. It serves the UI and API from one local server, and keeps BOM data in a local SQLite file.
